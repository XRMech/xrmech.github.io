What I Build for Fun

When I'm not on contract I'm usually building something computationally interesting.

StockMaster

Algorithmic trading platform with a political intelligence layer — detects information asymmetry by monitoring politician trading patterns against market movements. ML stack covers PyTorch LSTM and Transformer models (GPU/CUDA), XGBoost, scikit-learn, and Prophet for time series. 184 symbols across 11 tiers, circuit breakers, backtesting engine, paper trading via Alpaca API.

Python PyTorch CUDA FastAPI Celery PostgreSQL

DupliCleaner

Intelligent duplicate file detection for NAS storage. Goes well beyond checksum matching — perceptual hashing for near-duplicates, InsightFace/ArcFace for face recognition and clustering, CLIP for scene classification, YOLOv8 for object detection, EasyOCR for text. Organizes photos by EXIF date, GPS location, and event clustering. 341+ test cases, quarantine mode, full undo support.

Python CLIP YOLOv8 InsightFace Dear PyGui SQLite

HablaTranslate

Self-hosted real-time bidirectional Spanish/English speech translator running entirely on local hardware. WhisperX for ASR, Qwen3 via Ollama for context-aware translation (better idiom handling than Google Translate), Pyannote for speaker diarization, Silero VAD. Steady-state ~5GB VRAM. Vocabulary capture with SM-2 spaced repetition and Anki export.

Python WhisperX Ollama FastAPI WebSocket Local LLM

DyslexiaReader

KMM app for children ages 6–10 with dyslexia. Speech-to-text validates oral reading in real-time using Android SpeechRecognizer with Vosk offline fallback. Custom AlignmentEngine handles dyslexia-aware phonetic matching (b/d, a/e confusion tolerance) and tracks misread types. AI-generated stories via OpenAI, SM-2 spaced repetition for sight words, COPPA/GDPR compliant.

KMM Kotlin Jetpack Compose Vosk STT OpenAI
